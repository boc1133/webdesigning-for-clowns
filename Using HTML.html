How to use HTML

HTML stands for HyperText Markup Language

It is the basic structure of web coding. Think of it as the content of the webpage

This page is written in HTML

Using Atom

We will be using the Atom programme to write our code

Steps to follow

  1. Go to 'open' then 'New Folder' name a new folder.
  2. Right click new folder and add a file and name it 'index.html'
  3. type html return in index file
  4. Put 'title' in for web page name
  5. Save and then click on index.html file in finder (Chrome has to be default browser)
  6. Or right click on index.html (in atom) and then 'Copy full path' and then paste in browser
  7. Your page is now displayed in your browser. The title will be at the top of your page on the browser tab
