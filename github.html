How to use github to make your page public

  1. Get a github account
    1. Go to github
    2. Set up a new account
    3. Use the free version
    4. You will need to verify by an email link
  2. Make a new repository
    1. Click on new repository
    2. Give your site a name
    3. Decide if it is public or private
    4. Click create repository
  3. Load files
    1. Click on upload file
    2. Place files in the box
    3. Upload files
    4. Scroll down to GitHub Pages
    5. Change source to Master Branch and click 'Save'
  4. Your web page is now public at the address given
