Download these packages into Atom

  1. Go to 'Atom' tab and then 'Preferences'
  2. Then go to install
  3. Search for each of the following packages and install them
    • atom-beautify
    • atom-ternjs
    • autoclose-html
    • emmet
    • csslint
    • linter-jshint
    • pigments
    • language-ejs

For linter-jshint follow the instructions below

  1. Go to 'Preferences' under 'Atom' Tab
  2. Click on Preferences Tab
  3. Find 'linter-jshint'
  4. Click on 'Settings' Tab
  5. Tick box that says 'Lint inline Java Script'
